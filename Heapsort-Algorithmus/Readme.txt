> In diesem Projekt beschäftigen wir uns mit dem Sortierverfahren Heapsort und wie man ein Array aus Zeichenketten mit Heapsort aufsteigend sortierten kann.
 Zudem geht es um die Berechnung von Laufzeiten dieser Methode

## General Information
- Das Projekt beinhaltet 3 Klassen(Stringssort, RandomString ,Tester) 
- In Klasse StringsSort werden die Strings aufsteigend mit Heapsort Verfahren sortiert.
- In Klasse RandomString werden zufällige Strings erzeugt.
- In Tester Klasse wird die Laufzeit von Heapsort implementierung von n-mal Strings in einem 
Array ermittelt.
